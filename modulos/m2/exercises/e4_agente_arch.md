# Agente architect

Crea un agente especializado `@architect` con responsabilidadeas para definir:
- Arquitectura general del proyecto
- ADRs clave
- Convenciones de codificación
- Diagramas de arquitectura

Luego usa este agente para generar:
- Un diagrama de clases para el proyecto
- Un diagrama secuencia de requests
- Un diagrama deployment
- Un conjunto de ADRs iniciales basadas en el caso práctico
- Un documento de convenciones de codificación basado en las mejores prácticas para el stack tecnológico elegido

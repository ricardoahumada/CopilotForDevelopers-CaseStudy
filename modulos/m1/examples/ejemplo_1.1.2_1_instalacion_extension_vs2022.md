# Pasos: Instalación de GitHub Copilot en Visual Studio 2022

**Contexto de uso:** Procedimiento paso a paso para instalar la extensión de GitHub Copilot en Visual Studio 2022.

**Pasos de instalación:**

```
1. Abrir Visual Studio 2022
2. Ir a Extensions → Manage Extensions
3. En la barra de búsqueda, escribir "GitHub Copilot"
4. Seleccionar "GitHub Copilot" de los resultados
5. Hacer clic en "Download"
6. Cerrar Visual Studio para iniciar la instalación
7. El VSIX Installer se ejecutará automáticamente
8. Reiniciar Visual Studio 2022
```

**Requisitos previos:**

| Requisito | Versión mínima |
|-----------|----------------|
| Visual Studio 2022 | 17.6 o superior |
| Cuenta de GitHub | Con licencia Copilot activa |
| Sistema Operativo | Windows 10/11 o Windows Server 2019+ |

**Referencia oficial:** https://learn.microsoft.com/en-us/visualstudio/ide/ai-powered-intellisense

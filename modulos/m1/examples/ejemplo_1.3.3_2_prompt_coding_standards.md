# Prompt: Documentar Coding Standards en Constitución

**Contexto de uso:** Prompt para generar la sección de coding standards en constitution.md.

**Archivo:** `.specify/memory/constitution.md` (sección Coding Standards)

**Prompt para documentar estándares:**

```
@agent Genera la sección de coding standards para
.specify/memory/constitution.md incluyendo:

## Coding Standards
- Naming conventions por tipo de elemento
- Formatting rules (indentación, spacing)
- Documentation requirements
- Error handling patterns
- Async patterns
- Validation patterns

## Lenguajes
- C# (.NET 8)
- TypeScript (React)
- Python (FastAPI)
- SQL (T-SQL)

Para cada lenguaje, proporciona ejemplos específicos
de código correcto e incorrecto
```

**Contenido esperado:**

| Sección | Descripción |
|---------|-------------|
| Naming conventions | Reglas de nomenclatura |
| Formatting rules | Indentación, espaciado |
| Documentation | Requisitos de documentación |
| Error handling | Patrones de errores |
| Async patterns | Patrones asíncronos |
| Validation | Patrones de validación |

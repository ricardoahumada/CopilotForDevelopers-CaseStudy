# Prompt: Generar AGENTS.md para Proyecto Polyglot

**Contexto de uso:** Prompt para generar AGENTS.md en proyectos con múltiples tecnologías.

**Prompt:**

```
"Crea un archivo AGENTS.md para un proyecto con:
- Backend: Python con FastAPI, SQLAlchemy, PostgreSQL
- Frontend: TypeScript con React 18, Vite, Tailwind CSS
- Mobile: React Native con Expo
- DevOps: Docker, Azure DevOps
- Incluya Agent Skills para API generation, React components y database migrations"
```

**Características del proyecto polyglot:**

| Capa | Tecnología | Responsabilidad |
|------|------------|-----------------|
| Backend | Python/FastAPI | APIs REST |
| Frontend | React/TypeScript | Interfaz web |
| Mobile | React Native | Aplicación móvil |
| DevOps | Docker/Azure | Despliegue |

**Agent Skills sugeridos:**

- `api-generation` - Generación de endpoints FastAPI
- `react-components` - Componentes React reutilizables
- `db-migrations` - Migraciones de base de datos

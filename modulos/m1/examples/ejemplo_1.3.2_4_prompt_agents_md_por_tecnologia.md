# Prompt: AGENTS.md Específico por Tecnología

**Contexto de uso:** Prompt para generar versiones específicas de AGENTS.md según la tecnología del proyecto.

**Prompt para generar AGENTS.md por tecnología:**

```
"Genera una versión específica de AGENTS.md para:

1. Proyecto C# ASP.NET Core:
   - Incluir comandos .NET CLI
   - Estándares Microsoft C# Conventions
   - Entity Framework patterns

2. Proyecto Python:
   - Incluir comandos pip/uv
   - PEP 8 standards
   - Virtual environment setup

3. Proyecto React Native:
   - Incluir comandos npm/expo
   - TypeScript strict mode
   - React hooks best practices
```

**Comandos por tecnología:**

| Tecnología | Comandos principales |
|------------|---------------------|
| C# ASP.NET | `dotnet build`, `dotnet test`, `dotnet ef` |
| Python | `pip install`, `uv sync`, `pytest` |
| React Native | `npm install`, `expo start`, `npx jest` |

**Estándares por tecnología:**

| Tecnología | Estándar |
|------------|----------|
| C# | Microsoft C# Conventions |
| Python | PEP 8 |
| React Native | TypeScript strict, React hooks patterns |

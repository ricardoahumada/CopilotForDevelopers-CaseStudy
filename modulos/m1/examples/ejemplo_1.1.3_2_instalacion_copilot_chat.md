# Pasos: Instalación de GitHub Copilot Chat en VS Code

**Contexto de uso:** Procedimiento para instalar la extensión de chat interactivo de Copilot.

**Pasos de instalación:**

```
1. En el panel de extensiones
2. Buscar "GitHub Copilot Chat"
3. Instalar la extensión
4. Esto habilita la ventana de chat con Copilot (Ctrl+Alt+I)
```

**Uso del chat:**

- Abrir chat: `Ctrl+Alt+I`
- El chat permite hacer preguntas sobre código, solicitar explicaciones y generar código mediante conversación

# Pasos: Instalación de GitHub Copilot en VS Code

**Contexto de uso:** Procedimiento para instalar la extensión principal de GitHub Copilot en Visual Studio Code.

**Pasos de instalación:**

```
1. Abrir VS Code
2. Presionar Ctrl+Shift+X para abrir el panel de extensiones
3. Buscar "GitHub Copilot"
4. Seleccionar la extensión de GitHub
5. Hacer clic en "Install"
6. Reiniciar VS Code tras la instalación
```

**Extensiones complementarias recomendadas:**

| Extensión | Propósito | Atajo de activación |
|-----------|-----------|---------------------|
| GitHub Copilot | Sugerencias de código inline | Automático |
| GitHub Copilot Chat | Chat interactivo con IA | `Ctrl+Alt+I` |
| Spec Kit (opcional) | Desarrollo dirigido por specs | CLI |
